# 新勢力分會 · 追分儀表板｜副主席每週上傳 SOP

> 每週例會後，把 PALMS 匯出檔上傳一次，全分會夥伴重整頁面就看到最新成績。

## 一次性準備（只做一次）

1. 到 https://github.com 註冊一個免費帳號（有的話跳過）
2. 收灰哥寄來的「協作者邀請」信 → 點信裡的 **Accept invitation**
   （或到 https://github.com/geniusjohn0502/newforce-greenlight/invitations 接受）

## 每週上傳（3 步，約 1 分鐘）

1. 登入 GitHub 後，打開這個上傳連結：
   **https://github.com/geniusjohn0502/newforce-greenlight/upload/main/data**
2. 把本週的 **PALMS 匯出檔**（BNI 系統匯出的 `.xls` 檔）拖進頁面
   - 檔名不用改，系統會自動讀出日期
   - ⚠️ 匯出 PALMS 時，統計區間請設「**分會成立至今**」（十月起改設「滾動六個月」，屆時灰哥會提醒）
3. 頁面往下捲，按綠色的 **Commit changes** 按鈕

完成！夥伴打開下面這個網址、重新整理就是最新成績：
**https://geniusjohn0502.github.io/newforce-greenlight/**

## 常見問題

- **傳錯檔／傳重複了？** 再傳一次正確的即可，系統永遠顯示「檔名日期最新」的那份。
- **看不到 Commit 按鈕？** 確認你已登入、且已接受協作者邀請。
- **成績怪怪的？** 先確認 PALMS 匯出區間對不對（成立至今）；還是不對就跟灰哥說。
